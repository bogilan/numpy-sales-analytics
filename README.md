# Sales Analytics with NumPy

A portfolio project focused on sales data analysis using NumPy and Matplotlib.

## Project Overview

This project simulates 365 days of retail sales data for multiple products and performs business analytics calculations including:

- Total sales analysis
- Product performance ranking
- Moving averages
- Anomaly detection
- Seasonal trend simulation
- Sales visualization

The goal of this project is to demonstrate practical NumPy skills used in real-world data analytics workflows.

---

## Technologies Used

- Python
- NumPy
- Matplotlib

---

## Features

### KPI Analysis
- Total yearly sales
- Average daily sales
- Best/worst sales days
- Best/worst products

### Statistical Analysis
- 7-day moving average
- Standard deviation
- Anomaly detection

### Data Visualization
- Daily sales trends
- Product comparison charts

### Realistic Data Simulation
- Seasonal sales patterns using sine waves
- Reproducible random datasets
